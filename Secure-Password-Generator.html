Acceptable Use Policy

Purpose. This tool is provided for internal use to generate strong, cryptographically secure passwords and passphrases for business systems, accounts, and services.

Approved use. Generate passwords for work accounts, service credentials, and internal systems. The passphrase mode with EFF Diceware (7,776-word pool) is recommended for any credential you must memorise.

Data handling. This tool operates entirely client-side. No passwords, settings, or usage data are transmitted, logged, or stored. All generated values exist only in browser memory and are discarded when the tab is closed.

Prohibited use. Do not use this tool to generate credentials for systems outside the organisation, or share generated passwords via unencrypted channels (email, chat). Store credentials in the approved password manager only.

Integrity. Always use the copy distributed from the official internal source. Verify the SHA-256 file hash shown in the footer matches the published value before use on sensitive systems.

Questions or concerns? Contact IT Security via the internal helpdesk.
